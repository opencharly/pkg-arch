## 2026-07-02 — OpenCharly genesis

Moved to `github.com/opencharly/pkg-arch` (formerly `github.com/overthinkos/overthink-arch`) and restarted from a **single fresh initial
commit** — no prior history or tags were migrated. Part of the coordinated OpenCharly
genesis release `v2026.183.1359`: the container registry namespace moved to
`ghcr.io/opencharly` and every `@github` pin was repointed to the new org, the new
repo names, and the genesis tag. The changelog policy is now **one file per CalVer
release version** (`<YYYY.DDD.HHMM>.md`). Native Arch PKGBUILD for the `charly` binary; the `opencharly-git` package's old-name `conflicts`/`replaces` (`overthink-git`, `ov-git`) were dropped (tag-exempt).

See the `charly` superproject's `CHANGELOG/2026.183.1359.md` for the full narrative.
Pre-genesis history remains in the archived `github.com/overthinkos/overthink-arch` repository.
